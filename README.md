Real-Time Human Movement Detection System
Overview

This project implements a Real-Time Human Movement Detection System using Python, OpenCV, and Flask. The system captures live video from a webcam and processes the frames to detect human movement. When movement is detected, the system highlights the detected human with a bounding box and displays a detection alert.

The project is designed to be lightweight, efficient, and suitable for low-cost surveillance applications. It runs on standard hardware without requiring high computational resources.

Technologies Used

Python

OpenCV

Flask

HTML / CSS

Features

Real-time human movement detection

Bounding box around detected humans

Alert message when movement is detected

Web interface to start the detection process

Lightweight implementation that runs on standard systems

How to Run the Project

Install required libraries:

pip install opencv-python flask

Run the application:

python app.py

Open the browser and go to:

http://127.0.0.1:5000

Click Start Motion Detection to begin detection.
